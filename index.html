Hypergeometric Surprise Explorer

Explore Weaver's Surprise Index for a hypergeometric random variable Y ∼ Hypergeom(N, M, n), where: N = population size, M = number of “successes” in the population, and n = sample size without replacement. For each outcome y, the Weaver Surprise Index is SI(y) = Σ p(y')² / p(y).

Controls

N = population size (≤ 1000). M = number of “successes” in the population (0 ≤ M ≤ N). n = sample size (0 ≤ n ≤ N). Y counts the number of successes drawn.
Weaver SI interpretation (rules of thumb)
  • SI < 5: Not surprising
  • SI ≈ 10: Begins to be surprising
  • SI ≈ 1,000: Definitely surprising
  • SI ≈ 1,000,000: Very surprising
  • SI ≈ 10¹²: “Miracle!”
